# VERDEVIA — Portal Público de Carreiras (InHire Style)

Este projeto é o portal público de vagas do ecossistema VERDEVIA, projetado para permitir que candidatos se inscrevam em oportunidades de trabalho de forma ágil e segura.

## 🛡️ Compliance de Privacidade (LGPD & ANPD)

Desenvolvido sob o princípio de **Privacidade por Design (Privacy by Design)**, o portal não implementa login ou cadastro de senha para usuários normais/candidatos.

- **Conformidade com a LGPD**: Reduz a coleta e armazenamento de dados pessoais sensíveis e credenciais, minimizando riscos de vazamento de dados.
- **Fluxo Simplificado**: O candidato apenas visualiza as vagas e se candidata preenchendo um formulário direto de uma única etapa.
- **Termo de Consentimento**: O formulário de candidatura exige o aceite explícito de consentimento de uso de dados pessoais estritamente para o processo seletivo da vaga solicitada, em plena conformidade com a LGPD (Lei nº 13.709/2018).

---

## 🎨 Estética & Design (Apple Standard)

- **Layout Fluido**: UI sem dimensões em pixels fixos, se ajustando do mobile a telas ultra-wide.
- **Glassmorphism**: Efeitos translúcidos e mesh-gradients de alto contraste baseados em verde-esmeralda e azul-profundo.
- **Micro-interações**: Feedback instantâneo no envio da candidatura com animações de entrada e transição suaves (Framer Motion).

---

## 🚀 Como Iniciar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicialize o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
   Acesse em: `http://localhost:3003`
